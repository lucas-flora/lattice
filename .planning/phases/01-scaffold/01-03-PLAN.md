<task type="auto">
  <name>Task 1: Create Worker message protocol and handler</name>
  <files>src/engine/worker/protocol.ts, src/engine/worker/simulation.worker.ts, src/engine/worker/createSimulationWorker.ts, src/engine/worker/index.ts</files>
  <action>
  Create the type-safe Worker communication layer:

  1. `src/engine/worker/protocol.ts` -- Typed message protocol:
     - `WorkerInMessage`: Discriminated union with types `init`, `tick`, `stop`
       - `init`: `{ type: 'init' }`
       - `tick`: `{ type: 'tick' }`
       - `stop`: `{ type: 'stop' }`
     - `WorkerOutMessage`: Discriminated union with types `initialized`, `tick-result`, `error`
       - `initialized`: `{ type: 'initialized', generation: 0 }`
       - `tick-result`: `{ type: 'tick-result', generation: number, timestamp: number }`
       - `error`: `{ type: 'error', message: string }`
     - Export a `handleMessage(msg: WorkerInMessage, state: WorkerState): WorkerOutMessage | null` pure function
       - This is the testable core -- it takes a message and current state, returns a response
       - WorkerState: `{ generation: number, isRunning: boolean }`

  2. `src/engine/worker/simulation.worker.ts` -- Worker entry point:
     - Import `handleMessage` and types from protocol
     - Maintain mutable WorkerState
     - Listen for messages, call handleMessage, post results
     - Use `self as unknown as DedicatedWorkerGlobalScope` pattern for type safety
     - On 'init': reset state, respond with initialized
     - On 'tick': increment generation, respond with tick-result including timestamp
     - On 'stop': set isRunning to false

  3. `src/engine/worker/createSimulationWorker.ts` -- Factory for main thread:
     ```typescript
     export function createSimulationWorker(): Worker {
       return new Worker(new URL('./simulation.worker.ts', import.meta.url));
     }
     ```

  4. `src/engine/worker/index.ts` -- Re-exports:
     - Export types from protocol
     - Export createSimulationWorker

  CRITICAL: The worker file and protocol must have ZERO imports from react, next, three, or zustand. These are pure engine files.
  </action>
  <verify>pnpm tsc --noEmit passes; pnpm lint passes (engine isolation rule)</verify>
  <done>Worker message protocol defined with TypeScript types, handler extractable as pure function, Worker file created</done>
</task>
