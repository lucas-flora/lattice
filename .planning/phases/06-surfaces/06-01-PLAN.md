# Plan 06-01: Extended Commands, Controller Methods, and Store Fields

## Objective

Extend the engine and command infrastructure with all new commands and controller methods needed for Phase 6 surfaces. This includes simulation controls (stepBack, clear, speed, seek, status), cell editing (draw, erase, brushSize), and preset listing. Also extend stores with liveCellCount, speed, and brushSize fields.

## must_haves

- SimulationController gains: stepBack(), clear(), setSpeed(), seek(), getStatus(), getLiveCellCount()
- New sim commands: sim.stepBack, sim.clear, sim.speed, sim.seek, sim.status
- New edit commands: edit.draw, edit.erase, edit.brushSize
- New preset command: preset.list
- simStore extended with liveCellCount, speed fields
- uiStore extended with brushSize field
- EventBus extended with new event types: sim:speedChange, sim:clear, sim:stepBack, sim:seek, edit:draw, edit:erase
- wireStores updated for new events
- Tick diffs captured in CommandHistory before each tick (for reverse-step)
- liveCellCount computed on each tick

## Tasks

<task id="1">
Extend `src/engine/core/EventBus.ts` EngineEventMap with new event types:
- `sim:speedChange`: `{ fps: number }`
- `sim:clear`: `Record<string, never>`
- `sim:stepBack`: `{ generation: number }`
- `sim:seek`: `{ generation: number }`
- `edit:draw`: `{ x: number; y: number }`
- `edit:erase`: `{ x: number; y: number }`
</task>

<task id="2">
Extend `src/commands/SimulationController.ts` with new methods:
- `stepBack()`: undoes the last tick by restoring the previous generation's diff. Uses a tick history stack (circular buffer of sparse diffs captured before each tick). Emits `sim:stepBack`.
- `clear()`: zeros all cells in the grid (sets primary property to 0). Emits `sim:clear`.
- `setSpeed(fps: number)`: changes tick interval. 0 = max speed (use requestAnimationFrame semantics, interval of 0). Restarts tick loop if currently playing. Emits `sim:speedChange`.
- `seek(generation: number)`: navigate to a specific generation in history. If target < current, replay from reset. Emits `sim:seek`.
- `getLiveCellCount()`: count non-zero values in primary property buffer.
- `getStatus()`: returns `{ generation, liveCellCount, isRunning, activePreset, speed }`.
- Update `doTick()` to capture tick diff before applying tick (for stepBack), and include liveCellCount in sim:tick event.
- Store tick history as a circular buffer (max 1000 entries) of grid snapshots (or sparse diffs).
</task>

<task id="3">
Extend `src/store/simStore.ts`:
- Add `liveCellCount: number` (default 0) to SimState
- Add `speed: number` (default 10, meaning 10 FPS) to SimState
- Add `setLiveCellCount(count)` and `setSpeed(fps)` to simStoreActions
</task>

<task id="4">
Extend `src/store/uiStore.ts`:
- Add `brushSize: number` (default 1) to UiState

<task id="5">

- `sim.seek`: params `{ generation: z.number() }`, calls controller.seek(generation)
- `sim.status`: no params, returns controller.getStatus()
</task>
