# Plan 06-03: GUI Components: HUD, Controls, Preset Selector, Parameter Panel

## Objective

Build all GUI surface components: HUD (generation counter, live cell count), ControlBar (play/pause/step/reset/speed), PresetSelector dropdown, and ParamPanel sidebar. All buttons invoke commands through CommandRegistry.

## must_haves

- HUD shows: preset name, generation counter (tabular-nums), live cell count
- ControlBar with play/pause/step/step-back/reset/clear buttons using unicode symbols
- Speed slider with discrete FPS values (1, 5, 10, 30, 60, max)
- Timeline scrubber (horizontal slider for generation seek)
- PresetSelector dropdown listing all 6 built-in presets
- ParamPanel shows current preset parameters (read-only)
- All buttons invoke through CommandRegistry

## Tasks

<task id="1">
Create `src/components/hud/HUD.tsx`:
- Overlay in the top-left corner (absolute positioning, z-10)
- Shows: "Lattice" title (zinc-500, uppercase, tracking-wider), preset name (zinc-600), generation counter (green-400, tabular-nums, text-lg), live cell count (zinc-400)
- Reads from simStore: generation, liveCellCount, activePreset
- Updates reactively via useSimStore selectors
</task>

<task id="2">
Create `src/components/hud/ControlBar.tsx`:
- Horizontal toolbar at bottom-center of viewport
- Buttons: Play/Pause toggle (unicode: play triangle / pause bars), Step Forward (step arrow), Step Back (reverse arrow), Reset (reset arrow), Clear (X symbol)
- Speed slider: range input with discrete stops [1, 5, 10, 30, 60, 0] where 0 = "max"
- Labels for speed slider showing current FPS
- All buttons call `commandRegistry.execute("sim.play")`, etc.
- Play/pause button toggles based on simStore.isRunning
- Dark theme: zinc-800 background, zinc-200 text, green accent for active state
- Absolute positioning at bottom-center, z-10
</task>

<task id="3">
Create `src/components/hud/PresetSelector.tsx`:
- Dropdown in the top-right area of the screen
- Lists all 6 built-in presets from BUILTIN_PRESET_NAMES
- Selecting a preset calls `commandRegistry.execute("preset.load", { name })`
- Shows currently active preset as selected value
- Dark theme styling: zinc-800 background, zinc-200 text
- Absolute positioning top-right, z-10
</task>

<task id="4">
Create `src/components/panels/ParamPanel.tsx`:
- Side panel on right edge, slides in when isParamPanelOpen is true
- Toggle button visible in top-right area (gear icon or similar unicode)
- Shows current preset parameters (read-only):
  - Preset name
  - Grid dimensions (width x height)
  - Topology
  - Cell property definitions (name, type, default, role)
- Collapsible sections for each parameter group
- Dark theme: zinc-900 background, zinc-300 text
- Slide-in animation via CSS transition (transform: translateX)
</task>
